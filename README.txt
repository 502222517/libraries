1.grunt 文件夹为打包配置目录。(“grunt配置”文件夹修改为“grunt”)
2.src   为代码源目录，修改代码请从src修改。
3.当前目录为打包生成后目录。


grunt打包说明：
1.请注意将 “grunt配置”文件夹修改为“grunt”
2.下载nodejs,并安装。（地址百度）
3.安装 grunt cli 
   运行DOS，输入命令：npm install -g grunt-cli
   
4.进入grunt文件夹，运行DOS，输入命令：
   npm install 

5. 再次输入命令
   发布打包（会压缩）：grunt
   开发打包（不压缩，并开启watch监听）:grunt dev
